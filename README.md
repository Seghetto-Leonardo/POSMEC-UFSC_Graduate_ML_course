# POSMEC-UFSC_Graduate_ML_course
Scripts and analyses which I developed during 2026-Q2 for the Machine Learning &amp; Data Science course from UFSC|EMC's Graduate program.

The course covers the Section II from the book *Data-Driven Science and Engineering: Machine Learning, Dynamical Systems and Control* from Steven L. Brunton and J. Nathan Kutz, entitled *Machine Learning and Data Analysis*.

The repository is composed of two main folders: Regression and Classification. Each one of these contains the tasks and projects developed with the sole purpose of learning and visualizing the mathematical and conceptual topics presented during the course (each task/project contains its own README.md, aiming to support the learning process from whoever is interested).
